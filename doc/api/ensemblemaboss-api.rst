EnsembleMaBoSS API
==================

:doc:`ensemblemaboss-api-ensemble` is the object representing the UPMaBoSS simulation. It allows simulation tweaking, and execution. 
The execution, via the run() method, returns a :doc:`ensemblemaboss-api-ensembleresult` object which allows to plot and save results.
